About this server

Finally, a PvP server for the 90s! This is Project Enyo, a Minecraft server hosting pickup games on classic pvp maps. The server uses the PGM plugin to manage automatic matches.

  • Mumble: mumble.projectenyo.eu
  • IRC: #enyo @ quakenet.org
  • Discord: discord.gg/WKJkgdg

You can vote for the server on Planet Minecraft to help the server get more recognition.

Play

Once you join the server, you can watch the game as an observer. If teams are not full, you can type /join to play.

There are two map pools available, alpha and blitz. The alpha pool has the classics, while the blitz pool features maps with limited lives, often just one.

OPs are indicated by an orange * symbol in front of their usernames, and VIPs (if any) have a green *. If someone has a blue *, that means they are the author of the map! Current operators are Brottweiler and Noel.

Commands

  • /join — join the game
  • !message — shout to the other team
  • /votekick — start a votekick
  • /sit — sit down anywhere
  • /motd — show the message of today
  • /rules — view the rules
  • >implying

Rules

  • Don't be rude in chat
  • Do not grief or kill your own team
  • Do not use cheats
This server is run for the hell of it, not for profit. Playing is a privilege, not a right.
