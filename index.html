About this server

This is Project Enyo, a Minecraft server hosting pickup games on classic pvp maps.
The server uses the PGM plugin to manage automatic matches.

Play

Once you join the server, you can watch the game as an observer.
If teams are not full, you can type /join to play.
OPs are indicated by an orange * symbol in front of their usernames.
There is a Mumble server at mumble.projectenyo.eu, and a Discord server where I share news. It has a cool status bot.

Commands

Type /join to play and use !message to shout.
You can start a /votekick if someone behaves bad, and sit down with /sit.

Rules

  • Do not use cheats
  • Do not grief your own team
  • Don't be rude in chat

Also check out oc.tc!
