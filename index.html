Project Enyo

This is Project Enyo, a Minecraft server where you can play classic pvp maps anytime, anywhere.
The server uses the PGM plugin to manage automatic matches.

Play

To play, connect to pvp.projectenyo.eu with any Minecraft version.
There is also a mumble server at mumble.projectenyo.eu, and a Discord server where I share news. It also has a cool status bot.
OPs are indicated by an orange * symbol in front of their usernames.

Commands

Type /join to play and /g to speak in global chat.
You can start a votekick with /votekick , and sit down with /sit.

Rules

  • Do not use cheats.
  • Do not grief your own team.
  • Don't be rude in chat.
