About this server

Finally, a PvP server for the 90s! This is Project Enyo, a Minecraft server hosting pickup games on classic pvp maps.
The server uses the PGM plugin to manage automatic matches.
Use mumble.projectenyo.eu to talk with other players, and chat at #enyo @ quakenet.org. If you prefer Discord, it has a cool status bot and a bridge between both IRC and Mumble.

You can vote for the server on Planet Minecraft to help the server get more recognition.

Play

Once you join the server, you can watch the game as an observer.
If teams are not full, you can type /join to play.
There are two map pools available: the default alpha pool with the classics, and a blitz pool featuring maps with limited lives, often just one.
OPs are indicated by a gray @ symbol in front of their usernames.

Commands

Type /join to play and use !message to shout.
You can start a /votekick if someone behaves bad, and sit down with /sit.
Read the message of today with /motd and check the rules with /rules.

Rules

  • Don't be rude in chat
  • Do not grief or kill your own team
  • Do not use cheats

This server is run for the hell of it, not for profit. Playing is a privilege, not a right.
